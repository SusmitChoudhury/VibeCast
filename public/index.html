How It Works

Follow these steps to start your watch party.

For Hosts

  1. Enter your name and click "Create Room".
  2. Copy the Room Code (top right) and send it to friends.
  3. Paste a valid link:
    • ‚úÖ YouTube: (e.g. youtube.com/watch?v=...)
    • ‚úÖ Direct File: Any link ending in .mp4, .webm, .mkv
  4. Click "Load". The video will sync automatically.

For Guests

  1. Get the Room Code from the Host.
  2. Enter your name and the Code in the "Join Party" box.
  3. Click "Join Room".
  4. Wait! The video will start playing automatically once the Host loads it.
